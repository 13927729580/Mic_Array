This application reads and processes 8 digital microphones and generate 1 channel output of audio.
Data tream-out to USB audio: Channel Left:  MIC1(16bit)-MIC2(16bit)-MIC3(16bit)-MIC4(16bit)
                             Channel Right: MIC5(16bit)-MIC6(16bit)-MIC7(16bit)-BEAM(16bit) 
USB audio name: AUTONOMOUS MICROPHONE ARRAY, Number of Channel: 2, Sampling frequency:64Khz

List of ICs:
ST microcontroller Cortex M7: STM32F746IGT6
ST MEMS microphones (MP45DT02) 
Audio Signal Processing STA321MPL
CS43L22 Audio Codec Control
